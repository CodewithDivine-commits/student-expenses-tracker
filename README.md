# student Expenses Tracker 

A simple web application to help students track daliy expenses.

## Features
- Add expenses with title, amount, and category
- view all expenses in a organized list 
- calculate total spent
- categorize espense (Food, Data, Transportation, Rent and others)
- Delete individual expenses
- Data persistence with local storage 
- Responsive design for mobile and desktop

## Technologies used 
HTMLS
CSS 
Javascript(vanilla JS)

## How to use 
1. Enter expense details(title,Amount, category)
2. click "Add Expenses"
3. view your expenses in the list
4. see the total amount spent 
5. Delete expenses by clicking the "Delete"button
